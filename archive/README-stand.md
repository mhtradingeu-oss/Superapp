
# HairoticMen Stand Manager Plugin

## Features
- Stand Management with unique serial numbers
- Salon and owner information tracking
- Working hours management
- Inventory control
- WooCommerce integration
- Custom reporting

## Installation Requirements
- WordPress 5.8 or higher
- PHP 7.4 or higher
- WooCommerce 5.0 or higher

## Installation Steps
1. Download the plugin ZIP file
2. Install WooCommerce if not already installed:
   - Go to Plugins > Add New
   - Search for "WooCommerce"
   - Click "Install Now" then "Activate"
   - Complete the WooCommerce setup wizard

3. Install HairoticMen Stand Manager:
   - Go to Plugins > Add New > Upload Plugin
   - Choose the downloaded ZIP file
   - Click "Install Now"
   - Click "Activate Plugin"

4. Initial Setup:
   - Navigate to Stand Manager in WordPress admin menu
   - Configure general settings
   - Set up your first stand
   - Configure inventory sync with WooCommerce

## Post-Installation
1. Verify database tables are created correctly
2. Test stand creation functionality
3. Confirm WooCommerce product synchronization
4. Set up user permissions as needed

## Troubleshooting First-Time Setup
- Clear WordPress cache after installation
- Check PHP memory limit if experiencing issues
- Verify database permissions
- Ensure WooCommerce products are properly configured

## Usage

### Creating a New Stand
1. Navigate to Stand Manager > Manage Stands
2. Click "Add New Stand"
3. Fill in required information:
   - Stand name
   - Location
   - Salon details
   - Working hours
   - Initial inventory

### Managing Inventory
1. Go to Stand Manager > Inventory
2. Select a stand
3. Add/Remove products
4. Update stock levels

### Generating Reports
1. Access Stand Manager > Reports
2. Select report type:
   - Sales summary
   - Inventory levels
   - Commission reports
3. Filter by date range or stand

## Troubleshooting

### Common Issues

1. Data Not Saving
   - Check form submission
   - Verify database permissions
   - Clear browser cache

2. Reports Not Loading
   - Confirm WooCommerce connection
   - Check date range selection
   - Verify data exists for selected period
